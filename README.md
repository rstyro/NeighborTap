## 🏡 项目简介

**NeighborTap** 是一个创新的微信小程序项目，旨在成为本地生活服务的“毛细血管”。我们专注于解决那些 **“大平台看不上的、非标准化的、即时性的”** 小需求，通过构建一个基于地理位置的信任社区，让邻里之间的技能、时间和资源得到高效互换与价值流转。

项目名称源自 “Neighbor”（邻居）与 “Tap”（轻敲、利用），寓意着 **“轻敲邻居的门，即刻获得帮助”**，完美诠释了平台的核心理念。

## 💡 核心理念

*   **邻里互助**：重建现代都市中缺失的邻里关系，倡导“远亲不如近邻”的互助精神。
*   **技能变现**：让每个人的闲置技能（从PPT美化到家电维修）都能创造价值。
*   **时间换空间**：用你的空闲时间，为他人解决“小急事”，同时换取报酬或技能。
*   **解决“小急事”**：专注于那些小而急、标准平台无法覆盖的需求，如“帮忙取快递”、“临时看护宠物”、“半小时电脑调试”等。

## ✨ 功能特色

### 核心流程
- **📝 智能发布**：快速发布需求，支持技能互换、现金悬赏等多种形式，LBS自动匹配地理位置。
- **🔍 精准发现**：信息流与地图双模式浏览附近任务，智能筛选与搜索，快速找到心仪机会。
- **⭐ 信用生态**：双向评价体系构建信用画像，让每一次合作都成为下一次信任的基石。

### 特色模块
- **技能名片**：打造个人专属技能主页，展示才华，积累社区声誉。
- **急单高亮**：紧急需求优先展示，快速响应解决燃眉之急。
- **邻里圈子**：分享互助故事，参与本地话题，增强社区归属感。
